HTML & CSS

This Single-Page CV Website is a modern, interactive online resume built exclusively using HTML, CSS, and JavaScript. The site does not navigate to new HTML pages but instead dynamically updates its content by toggling active classes. This approach ensures a seamless user experience, offering smooth transitions between sections while maintaining high performance.

Features:

  1. Single-Page Layout:

    All information is organized into distinct sections: About Me, Skills, Experience, Education, and Contact. Navigation is implemented with clickable menu items that scroll or display the corresponding section dynamically.

  2. Dynamic Class Activation:

    Instead of loading new pages, clicking on a navigation item adds an active class to the targeted section and hides the others. Smooth transitions between sections are achieved using CSS animations.

  3. Fully Responsive Design:

    Designed with a mobile-first approach, ensuring optimal display on all devices. CSS media queries adjust layout and font sizes to enhance readability across screen sizes.

  4. Stylish Visuals:

    Clean and minimalistic styling with CSS for a professional look. Subtle hover effects and button animations provide an engaging user interface.

  5. Interactive JavaScript Features:

    Scroll animations that highlight the section currently in view. Form validation for the contact section ensures user inputs are correctly formatted.

  6. Benefits:

    Lightweight and Fast: No need for server requests or external frameworks. User-Friendly: Provides a smooth, uninterrupted browsing experience. Easy Customization: The structure is modular, making it easy to update content or add new sections.
This Single-Page CV Website demonstrates how pure front-end technologies can create professional, user-friendly websites with modern functionality and aesthetics.
